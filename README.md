# IADvogado

Projeto desenvolvido na disciplina de Inteligência Artificial com o objetivo de simplificar textos jurídicos utilizando modelos de linguagem.

## 📌 Descrição
O projeto propõe uma ferramenta capaz de transformar textos jurídicos complexos em versões mais simples e acessíveis, utilizando modelos de linguagem (LLMs).

## 📂 Estrutura
- `dataset/`: dados utilizados no projeto  
- `notebooks/`: análise exploratória  
- `src/`: código do projeto  
- `relatorio/`: artigo entregue  

## 🔎 Tecnologias
- Python  
- Pandas  
- Ollama  
- Tesseract OCR  
- Streamlit  

## 📊 Objetivo
Facilitar o entendimento de textos jurídicos para pessoas sem formação na área.

## ⚠️ Observação
Este sistema não substitui um advogado.
